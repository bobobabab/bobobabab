# Cli使用须知

# 启动web服务器

可以使用Npm或Yarn

npm i --save @jzjy/cli

或者

yarn add @jzjy/cli

或者全局安装

npm i -g @jzjy/cli

WARNING

建议全局安装本模块，可以在项目根目录进行快速开发组件及测试

# 启动web服务

快速启动静态html服务以及vue组件（支持热加载）

# 参数

参数 别名 描述 默认值
--help -h 帮助
--port -p 端口 8000
--proxy -P 代理请求地址
--open -o 是否自动打开浏览器 false
--vue -i 静态页面或者vue组件 HTML

# 示例

jzjzy serve <path>

静态HTML文件

# 当前目录为工程根目录
jzjy serve ./packages/ -P 'http://pigx-gateway:9999' -o -p 3000

VUE组件工程

jzjy serve ./packages/verify/example -P 'http://pigx-gateway:9999' -o -p 3000 -i

TIP

如若当前为vue工程，只需要一个vue文件和一个一个index.js作为入口，path必须指向入口文件，可以参考以下简单例子

# 目录结构

- lib
  - test.vue
  - index.js

# 组件test.vue

<template>
  <div id="test">
    // ...
  </div>
</template>

# 入口文件index.js

import Test from './Test.vue'

Test.install = Vue => {
  Vue.component(Test.name, Test)
}

if (typeof window !== 'undefined' && window.Vue) {
  window.Vue.use(Test)
}

export default Test
